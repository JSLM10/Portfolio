PROYECTOS

Página de compras

Tecnologías usadas: C#, ASP.NET, MVC, ADO.NET, SQL Server, AJAX, jQuery

Este proyecto es una tienda en línea desarrollada con el framework ASP.NET y el patrón de diseño MVC.
Permite a los usuarios agregar productos a un carrito de compras y luego realizar pagos a través de PayPal, en un futuro se espera poder vincularlas con otras aplicaciones bancarias que permitan el pago.

Gestor de Biblioteca

Tecnologías usadas: Java,NetBeans, MySQL, iTextPDF

Es un sistema de gestión de bibliotecas que permite a los bibliotecarios realizar las tareas como:
Agregar, eliminar y actualizar los libros
Registrar, modificar y eliminar datos de usuarios;
Registrar, controlar fechas y generar informes de los mismos en PDF

Gestor de calzado

Tecnologías usadas: Java, MySQL, Corel Draw, Jaspersoft

El Sistema de Ventas de Calzados está desarrollado en Java SE 8 con MySQL. Gestiona empleados, productos y ventas, generando reportes en texto, Excel y PDF. Usa JasperSoft Studio para comprobantes y gráficos en Graphics2D. Incorpora una GUI interactiva con animaciones y filtrado avanzado.
